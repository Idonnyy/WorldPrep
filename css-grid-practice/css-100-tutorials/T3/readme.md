# T3 - Two Column Layout

This project demonstrates a basic two-column layout using CSS Grid.

## Concepts Practiced
- display: grid
- grid-template-columns
- Fraction units (fr)
- Sidebar and content positioning
- Basic page layout structure

## Goal
Create a layout containing:
- A sidebar (25% width)
- A main content area (75% width)

## Files
- index.html
- style.css

## Notes
This layout is a foundational grid exercise used to understand column sizing and basic grid containers before moving into more advanced responsive layouts.
